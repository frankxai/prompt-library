# extract_wisdom

Extract the deepest wisdom, surprising ideas, actionable insights, key quotes, mentioned habits, surprising facts, references, and concrete recommendations from any input — paper, transcript, book, conversation. Output is structured into eight named sections in fixed order, each item terse and active-voiced. The classic "Fabric" pattern by Daniel Miessler (MIT), adapted here with brand-voice guards and the standard frontmatter contract. Use when you need to mine long content for compounding signal.
